# 🕯️ NotGone — I'm Not Gone.

> **“我想在一个人的世界里，也能被这个世界确认我还在。”**

NotGone 是一个专为 25-45 岁独居人群设计的“生命存在感系统”。它不是生硬的健康监测，而是一种通过每日 5 秒钟的「生命体征」记录，感受社会温度的温柔链接。

## 🌟 产品愿景 (Vision)
它的存在意义不是检测“死没死”，而是每天温柔地确认：**“你还在，这个世界在看着你。”**

## ✨ 核心特性 (Features)
- **生命温度打卡**：通过圆环长按交互，记录 0-100 的生命体征。
- **温柔语义映射**：不显示数字，将数据转化为“偏冷”、“温暖”、“炙热”等感性文案。
- **守望者系统 (Watchers)**：分级预警机制，当用户处于“失联”边缘，系统按阶梯向亲友/互助者发送邮件。
- **隐私至上**：无登录机制，仅基于设备的匿名 ID，不存储任何隐私文本。

## 🛠️ 技术架构 (Tech Stack)
- **Mobile**: React Native + Expo
- **Backend**: AWS Lambda (FastAPI) + API Gateway
- **Database**: DynamoDB
- **Notification**: AWS SES (Simple Email Service)
- **Infrastructure**: Serverless Architecture
